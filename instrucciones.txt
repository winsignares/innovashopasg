INSTRUCCIONES DE COMO CORRER EL PROYECTO:

SI NO SE TIENE EL PROYECTO INSTALADO:
1. clonar el repositorio - git clone (ruta del repositorio)
2. abrir cmd o git bash en el directorio del proyecto clonado
3. en el directorio crear la carpeta virtual - virtualenv venv
4. abrir la carpeta venv/Scripts en el cmd - cd /venv/Scripts
5. hacer activate.bat o source activate o activate (la que sirva)
6. volver al directorio raiz - cd ../ (dos veces)
7. instalar las dependencias - pip install -r requirements.txt
8. ejecutar el app.py - py app.py

SI SE TIENE EL PROYECTO INSTALADO
paso 4, 5, 6 y 8

INSTRUCCIONES DESPUÉS DE INSTALACIÓN:

El sistema necesita una conexión activa de los servicios apache y mysql, mantener activas estas conexiones.

Por comodidad de prueba se incluye un archivo de extensión .SQL en la carpeta con datos de prueba que pueden ser usados para probar el sistema en su totalidad.
Este archivo puede ser importado en el administrador phpMyAdmin de la base de datos de mysql.
